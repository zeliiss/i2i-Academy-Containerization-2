# i2i-Academy-Containerization-2

This repository contains the source code and configuration files for the Containerization and Docker assignment.

## Project Overview
The objective of this project is to deploy a static webpage using Docker and `docker-compose`. The environment was successfully tested on a local machine and subsequently deployed to a Google Cloud Platform (GCP) Virtual Machine.

## Repository Contents
- `docker-compose.yml`: Configuration to run the Nginx web server container.
- `index.html`: Custom HTML page served by the container.
